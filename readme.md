#  Student Management System

A sleek, full-stack Student Management Portal featuring a modern **dark-blue** user interface. This single-page application allows users to perform full CRUD (Create, Read, Update, Delete) operations on a MySQL database through Python Flask API.

##  Features

* **Create:** Add new student records (Roll Number, Name, CGPA, Admission Date).
* **Read:** View all existing students in a dynamically updated, responsive table.
* **Update:** Modify specific fields (First Name, Last Name, CGPA, or Admission Date) for existing students.
* **Delete:** Remove student records securely from the database.
* **Modern UI:** A stunning black-and-blue gradient background with a frosted glass container, hover animations, and custom scrollbars.
* **RESTful API:** Clean separation of concerns between the HTML/JS frontend and the Python backend.

##  Tech Stack

* **Frontend:** HTML5, CSS3 (Glassmorphism Theme), Vanilla JavaScript (Fetch API)
* **Backend:** Python 3, Flask, Flask-CORS
* **Database:** MySQL, `mysql-connector-python`


### Prerequisites

You will need the following installed on your system:
* [Python 3.x](https://www.python.org/downloads/)
* [MySQL Server](https://dev.mysql.com/downloads/installer/)
